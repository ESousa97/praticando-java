# Security Policy

> Arquivado: este repositório não está mais ativo e permanece público apenas para fins de estudo.
> Não há garantia de resposta, revisão, correção ou SLA para reportes de segurança.

## Supported Versions

| Version | Supported |
|---|---|
| none | no |

## Reporting a Vulnerability

Para reportar vulnerabilidades, envie um e-mail para:

- security@enoquesousa.dev

Inclua, quando possível:
- descrição do problema
- passos para reprodução
- impacto potencial
- sugestão de mitigação

## Response SLA

- Não aplicável enquanto o repositório estiver arquivado.

## Responsible Disclosure

- Não publique detalhes da vulnerabilidade antes de alinhamento com o mantenedor.
- Após a correção, um aviso pode ser publicado no `CHANGELOG.md`.
